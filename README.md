🔴 Smart Traffic Light Control System
📌 Overview

A sensor-based intelligent traffic light system that adjusts signal timing based on vehicle presence to improve traffic flow efficiency.

🎯 Purpose

Reduce traffic congestion

Automate signal control

Improve road efficiency

⚙️ Components Used

Microcontroller (Arduino / ESP32)

IR Sensors

Ultrasonic Sensors

LEDs (Traffic Signals)

Breadboard & Wiring

🔧 Working Principle

IR/Ultrasonic sensors detect vehicle presence

System analyzes traffic density

Signal timing adjusts dynamically

Lights change automatically based on conditions


💡 Key Features

Real-time traffic detection

Adaptive signal timing

Fully automated system
